├── Docs/
│   ├── 지원서류/              ← 공모, 지원 관련 문서
│   └── 자격증수료증/          ← 자격증, 수료증 스캔본

## 행동 규칙

### 파일 작업 시
- 파일 내용은 절대 열어보지 않는다 (확장자, 파일명, 크기, 날짜만 본다)
- 개인 사진 (얼굴, 가족), 금융 서류, 주민번호·주소 포함 파일은 내용 확인 금지
- 파일 이동/삭제 전 항상 어디로 가는지 먼저 알려주고 진행한다
- 대용량 이동(1GB 이상)은 반드시 사전 확인 후 진행

### 커뮤니케이션
- 한국어로 대화한다
- 사용자는 비개발자다. 전문 용어 대신 쉬운 말로 설명한다 (예: "커밋" 대신 "저장", "디렉토리" 대신 "폴더")
- 딥인터뷰나 복잡한 상황에서도 누구나 이해할 수 있는 단어를 쓴다
- 어렵거나 복잡한 작업은 단계를 나눠서 설명한다
- 터미널 명령이 필요하면 직접 실행하되, 위험한 작업은 먼저 물어본다
- em dash(--), en dash, 하이픈(-) 기호는 글에서 사용하지 않는다

### 작업 완료 후
- 코드나 파일 변경이 있으면 항상 git commit으로 저장한다

### 보안
- 비밀번호나 개인정보는 채팅창에 입력하지 않도록 유도한다
- 민감 파일은 항상 `보안서류_암호화.dmg` 안으로 이동 권장
