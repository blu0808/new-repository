### 지향점 및 가치관
1. **느슨한 연결** — 말이 많지 않아도 같은 공간에서 각자의 속도로 집중
2. **생활 속 문화·취미의 일상화** — 북클럽, 뜨개모임, 워크샵, 음악회, 요가
3. **장소 정체성** — 서촌이라는 지역성을 브랜드 핵심 자산으로 활용

### 기본 현황

### 스토리 하이라이트
- 2026빙수 / 2025빙수 / 2024빙수 / 2023빙수 (연도별 아카이빙)
- 롯지가을방학 / Tokyo! / 카페이용안내 / Lodge / 그래놀라 텍배 / lodge story

### 브랜드 성격
부부가 운영하는 소규모 동네 카페. 화려한 인테리어나 특별한 컨셉 없이 **빛이 잘 드는 공간, 계절마다 달라지는 창밖 풍경, 정직한 커피와 베이크드 치즈케이크**가 핵심. 카페 이용 안내를 손글씨로 쓰고, "머신 잘 고쳤습니다 ㅠㅠ"처럼 솔직하게 운영 상황을 공유하며, 팬이 그려준 공간 일러스트를 리포스팅하는 방식으로 소통.

### "번화가 아닌데 팔로워 1.3만"의 이유

**1. 시간의 축적이 브랜드가 된다**
9년 운영, 1,045개 게시물. 특별히 자극적이지 않은 콘텐츠가 쌓이며 "정말 좋은 곳"으로 인식 진화.

**2. 빙수 시리즈 = 연간 팔로워 유입 엔진**
매년 여름 자연 확산되는 빙수 콘텐츠 → 왔다가 팔로우 → 다음 해에 다시 오는 사이클 반복.

**3. 그래놀라 택배 = 오프라인 한계 극복**
직접 오기 어려운 지역 사람도 택배로 경험 구매 → 구매자가 팔로워로, 팔로워가 구매자로.

**4. 옆 공간 LOG = 협업 팔로워 유입 구조**
마켓, 기획전 개최 시 참여 브랜드·작가들이 각자의 팔로워에게 lodge190을 노출.

**5. 소통 방식이 다르다**
손글씨 이용 안내, 솔직한 운영 공지, 팬 일러스트 리포스팅 → 브랜드를 사람처럼 느끼게 함.

### 온수 참고 인사이트
> "공간의 감도 × 시간의 일관성 × 수익 다각화"
- 팔로워가 위치를 보고 오는 게 아니라 계정을 보고 온다
- 시즌 메뉴 + 온라인 판매 + 공간 확장 = 수익 다각화 모델
- **인스타그램 비주얼 방향: 계절과 빛을 담은 공간 사진**

### 브랜드 성격
카페테리아를 표방하지만 실제로는 **동네 문화 플랫폼**에 가깝다. 35개 게시물 안에 담긴 기획의 종류:

|--------|------|
| 작심삼월클럽 | 3개월짜리 창작 클럽 |

### 콘텐츠 기획의 특이점

**1. 비주얼 언어가 완전히 자체적**
사진 거의 없음 + 거친 드로잉 + 손글씨 + 마커 일러스트 → 강력한 아이덴티티 형성.

**2. 이벤트 기획에 유머가 있다**
"맥요일", "잔걸음배 팔씨름대회", "절반절반 데이" → 웃기지만 진심 → 자발적 공유(바이럴) 동력.

**3. 공간이 창작자들의 무대**
클럽 멤버 전시, 연극 단독 개최, DJ 이벤트 → 각 창작자의 팔로워가 잔걸음으로 유입.

### 핵심 수치
게시물 35개로 팔로워 1,197명 = 게시물 1개당 평균 34명. 세 곳 중 게시물 대비 팔로워 효율이 가장 높음. 매 게시물의 밀도와 기획력이 높다는 증거.

### 온수 참고 인사이트
> "기획의 밀도로 승부하는 실험형 공간"
- 같은 동네(녹번), 좁은 공간인데도 사람이 많이 오간다 (지인 후기)
- 온수와 비교: 잔걸음은 유머·실험적, **온수는 더 차분하고 정제된 에너지**
- 프로그램 다양성과 기획력은 참고할 것

# 4. @elak.coffee — 에락

### 기본 현황

| 계정명 | @elak.coffee |
| 브랜드명 | 에락 (愛樂) |

| 위치 | 서울 중구 청구로22길 36 1층 (신당동) |
| 팔로워 | 1.6만 명 |
| 게시물 수 | 1,052개 |
| 팔로우 수 | 0명 (단방향 발신 채널) |
| 온라인 판매 | 네이버 스마트스토어 운영 |

### 바이오
> "슬픔보단 사랑을, 놓쳐버린 것들을 찾는곳"

### 브랜드 성격
카페지만 커피 자체보다 **커피를 둘러싼 감정과 이야기**를 전면에 내세운다. 매달 원두를 소개할 때 감정적 테마를 먼저 설정하고, 그 감정과 어울리는 원두를 큐레이션한다. 사장님이 매일 카페 일지, 에피소드, 생각을 진심으로 기록. DJ 이벤트에서 이웃 어르신이 춤추는 장면을 전면에 내세움 — 세련됨과 동네 뿌리내림의 공존.

### 비주얼 언어
- 사진 위주, 다큐멘터리 감성
- 연출하지 않은 것처럼 보이는 연출: 골목, 자연스럽게 앉은 사람들, 춤추는 노인, 아이, 강아지
- 흑백 사진 간간이 혼합
- 빨간 벽 + 노란 조명의 외관이 강력한 공간 아이덴티티
- 90년대 필름 감성

### 콘텐츠 기획의 핵심 구조

**원두 큐레이션 = 감정 큐레이션**
상품 설명이 아니라 에세이로 씀. "이 커피를 왜 지금 당신이 마셔야 하는가"에 대한 이야기.

**팔로우 0 = 브랜드 선언**
받지 않고 주기만 하는 계정 → 일방향 발신이 오히려 신뢰를 만듦.

### 직접 방문 후기 (치기님)
- 체리우드 느낌과 아늑함이 좋았음
- 의자가 편한 건 아닌데 분위기와 사람냄새 나는 무드가 그걸 이겼음
- 동네 단골집 같은 느낌, 나만 알고 싶은 공간
- 사장님이 매일 올리는 일지에서 진심과 철학이 느껴짐
- 커피 매달 라인업이 바뀌는 것도 좋았음

### 온수 참고 인사이트
> "감정과 철학을 파는 정서 브랜드"
- **인스타그램 방향: 에락처럼 자연스럽게 있는 사람들**
- 장사를 하지만 장사처럼 느껴지지 않는 것 = 온수가 지향하는 "사람냄새"
- 운영하다 보면 자연스럽게 나오는 철학을 글로 기록하는 것

### 프로그램 구성

**워크샵** (음악/사운드, 비주얼/코딩, AI/기술, 사진/영상, 글쓰기/문학, 도예/조형, 기획/포트폴리오)

### 운영 방식

**튜터 기반 큐레이션 구조**
각 분야 독립 창작자를 '튜터'로 섭외해 프로그램 기획. 튜터는 강사이자 큐레이터, PSO는 공간·홍보·운영 인프라를 제공하는 플랫폼 역할. 튜터의 팔로워가 PSO로 자연 유입됨.

**서포터즈 시스템**
- 모집 인원: 2명 (2기 기준)
- 활동비: 월 10만원
- 혜택: 워크샵 무료 참관 + PSO 굿즈
- 역할: 워크샵 카드뉴스 제작, 현장 관리, 신규 콘텐츠 기획

**아카이빙 문화**
모든 워크샵, 이벤트, 튜터 인터뷰를 웹사이트 아카이브에 기록. 단순 후기가 아닌 브랜드 자산으로 축적.

### 온수 참고 인사이트
> "커뮤니티 플랫폼으로서의 공간"
- **튜터 섭외 구조**: 치기님이 모든 클래스를 직접 하지 않아도 됨. 각 창작자 튜터 섭외 → 튜터 팔로워 유입.
- **OneDayWeekMonth**: 카페 손님 vs 멤버 문제를 하루/일주일 이용권으로 해결. 경계를 명확하게 만드는 구조.
- **서포터즈 시스템**: 소규모 운영 인프라 구축 방법.
- 워크샵 → 동아리 → 공간대여 → 이벤트가 유기적으로 연결된 생태계 설계.

# 6. 보틀팩토리 (BOTTLE FACTORY)

### 기본 현황
| 항목 | 내용 |
|------|------|
| 계정명 | @bottle_factory |
| 대표 | 정다운 |
| 위치 | 서울 서대문구 홍연길 26, 1층 (연희동) |
| 팔로워 | 약 1.1만명 |
| 게시물 수 | 1,028개 |

### 브랜드 철학
> "Sustainable lifestyle design company"
> "질문을 던지고 실험하고 가능성을 검증한다"

모든 사업은 먼저 질문을 만들고, 직접 실험한 후, 검증된 솔루션을 확장하는 방식.

### 위치 및 공간 구성
- **연희동 보틀라운지 (메인):**
  - 1F: Coffee Bar + Less Waste Shop + Refill Station + Lounge + Vegan Grocery
  - B1: 보틀 그라운드 (커뮤니티 활동 공간, 대관 가능)
- **방배 숲의자리 보틀라운지 (지점)**

### 수익 모델
| 수익원 | 내용 |
|--------|------|
| 카페 음료·음식 판매 | 비건 디저트, 음료, 간단한 식사류 |
| 제로웨이스트 숍 판매 | 자체 제작 굿즈 (보자기, 빵주머니, 리턴미 컵 등) |
| 리필 스테이션 | 세제·샴푸 등 생활용품 무포장 리필 판매 |
| 보틀클럽 앱 (B2B) | 전국 300여 제휴 가게 대상 가입비·쿠폰 수수료 |
| 리턴미 솔루션 (B2B) | 다회용 컵 납품·대여·세척 서비스 |
| 공간 대관 | 보틀 그라운드(B1) 외부 대관 |
| 프로젝트·컨설팅 (Bottle Lab) | 지자체·기업 협력 제로웨이스트 프로젝트 기획·운영 |

### 주요 프로그램

**유어보틀위크 (Your Bottle Week)**
2018년부터 매년 11월, 연희동 일대에서 개최 (2025년 기준 7회). 동네 상점 50여 곳과 함께하는 지역 기반 생활 실천 페스티벌.

**채우장 / 살림장 / 바꾸장**
제로웨이스트 마켓, 중고·교환 마켓, 대안 경제 장터.

**다시살림학교**
살림·요리·수리·돌봄·생태 5개 영역, 2주 과정.

### 운영 방식

**실험→검증→확장 순환**
2016년 팝업 카페 실험 → 보틀라운지 오픈 → 보틀클럽 앱 → 리턴미 B2B 확장. 처음부터 완벽한 모델이 아니라 작게 실험하고 검증한 후 확장.

**지역 생태계 구축**
연희동을 거점으로 동네 소상공인들과 협력 네트워크 구축. 연희동을 '제로웨이스트 동네'로 브랜딩. 서대문구청 협력.

**B2G 활용**
환경부, 지자체(서대문구청, 군산시 등), 기업(LUSH 등)과 협력. 정부 제도(탄소중립실천포인트)까지 연동.

### 온수 참고 인사이트
> "사회적 가치 + 비즈니스가 일치하는 복합 수익 구조"
- **같은 서대문구** — 연희동에서 이미 작동하는 모델. 홍은동도 가능.
- **사회적 가치 언어화**: 온수의 "창작자 고립 해소, 동네 창작 생태계 형성"을 지원사업·지자체 협력 언어로 다듬으면 B2G 가능.
- **1층 공개 + 지하 커뮤니티** 구조: 공간이 더 크다면 카페 손님과 멤버 공간 분리 방법.
- **연간 동네 이벤트**: 유어보틀위크처럼 홍은동 일대 상점들과 함께하는 연간 기획 가능.
- 처음부터 완벽하지 않아도 됨 — 실험→검증→확장.
